# 🏍️ Rolê de Quinta - Série RQ 2026

Landing page para venda de moletons personalizados do grupo "Rolê de Quinta".

## 🚀 Funcionalidades

- Vitrine de cores (Vermelho, Rosa/Pink, Azul, Verde, Preto)
- Tabela de medidas responsiva
- Regras de compra em destaque
- Passo a passo para realizar pedido
- Botão para copiar chave PIX
- Link direto para WhatsApp de suporte
- Integração com Google Forms

## 🛠️ Tecnologias

- HTML5
- CSS3 (variáveis, Grid, Flexbox)
- JavaScript puro
- Google Fonts (Oswald + Roboto)

## 📁 Estrutura

## 🌐 Deploy (GitHub Pages)

1. Faça push para um repositório no GitHub
2. Vá em **Settings > Pages**
3. Selecione a branch **main** e salve
4. Acesse: `https://seu-usuario.github.io/role-de-quinta/`

## 📝 Personalização

### Alterar link do Google Forms
Edite o arquivo `index.html` e procure por:
```html
<a href="https://forms.google.com/SEU_FORM_AQUI" ...>
```

### Alterar chave PIX
Edite o arquivo `js/main.js` e procure por:
```javascript
const chave = '(41) 99968-4188';
```

### Alterar número do WhatsApp
Edite o arquivo `index.html` e procure por:
```html
<a href="https://wa.me/5541999684188" ...>
```
